# Job Intelligence Platform

An intelligent job monitoring and alert system that tracks job postings from the world's top 100 product-based companies and delivers personalized alerts based on user-defined criteria.

## 🎯 What It Does

- **Monitors 100+ Product Companies** — Tracks job postings from FAANG, SaaS, FinTech, and India-based product leaders
- **Smart Job Matching** — Filters roles by experience, skills, location, and job type using rule-based matching
- **Real-Time Alerts** — Sends email notifications instantly when jobs match your criteria
- **Zero Spam** — Content-based deduplication ensures you never get the same job alert twice
- **Reliable Detection** — Tracks job lifecycle (new, active, closed) across career page changes

## 🚀 Why This Exists

Job seekers targeting product-based companies face:
- ❌ Manual monitoring of 100+ career pages (impossible)
- ❌ Job portals flooded with irrelevant postings
- ❌ Career pages with no smart alert functionality
- ❌ High-signal roles getting missed

This platform solves all of it.

## 🏗️ Tech Stack

**Backend:** FastAPI + Python  
**Database:** PostgreSQL + Redis  
**Scraping:** Playwright + BeautifulSoup  
**Scheduling:** Celery + APScheduler  
**Deployment:** Docker + AWS (ECS/EKS)  
**Testing:** Pytest + unittest
